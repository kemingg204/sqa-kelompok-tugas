# Software User Documentation
## CV Rofile Chetose - Sistem Penjualan Kursi

---

## DAFTAR ISI

1. Panduan Admin
2. Panduan Manager
3. Panduan Kasir
4. Panduan Staff

---

## 1. PANDUAN ADMIN

### 1.1 Login Admin
1. Buka halaman login
2. Masukkan username: `admin` dan password: `admin123`
3. Klik Login

### 1.2 Menambah Kursi Baru
1. Klik menu "Kelola Kursi"
2. Klik tombol "Tambah Kursi"
3. Isi form (kode, nama, kategori, harga, stok)
4. Klik "Simpan"

### 1.3 Mengedit Kursi
1. Klik tombol "Edit" pada kursi yang ingin diubah
2. Ubah data yang diperlukan
3. Klik "Simpan"

### 1.4 Menghapus Kursi
1. Klik tombol "Hapus" pada kursi
2. Klik "Ya" pada konfirmasi

---

## 2. PANDUAN MANAGER

### 2.1 Login Manager
1. Masukkan username: `manager` dan password: `manager123`

### 2.2 Melihat Laporan
1. Klik menu "Laporan"
2. Pilih filter periode (Hari ini / 7 hari / Bulan ini)
3. Lihat grafik dan total pendapatan

### 2.3 Export Laporan
1. Klik tombol "Export Excel" untuk file .xlsx
2. Klik tombol "Export PDF" untuk file .pdf

---

## 3. PANDUAN KASIR

### 3.1 Login Kasir
1. Masukkan username: `kasir` dan password: `kasir123`

### 3.2 Melakukan Transaksi
1. Pilih kursi yang akan dibeli
2. Input jumlah
3. Klik "Tambah ke Keranjang"
4. Jika selesai, klik "Checkout"
5. Pilih metode pembayaran: Cash / Debit / QRIS
6. Klik "Proses"

### 3.3 Transaksi Cash
- Masukkan jumlah uang customer
- Sistem akan menghitung kembalian

### 3.4 Transaksi Debit
- Klik "Proses Debit"
- Tunggu konfirmasi dari mesin EDC

### 3.5 Transaksi QRIS
- Scan QR Code yang muncul
- Tunggu notifikasi pembayaran

---

## 4. PANDUAN STAFF

### 4.1 Login Staff
1. Masukkan username: `staff` dan password: `staff123`

### 4.2 Melihat Stok Kursi
1. Klik menu "Stok Kursi"
2. Lihat tabel stok (kode, nama, stok)
3. Bisa cari kursi menggunakan kolom pencarian
