# Gray Box Testing – CV Rofile Chetose

**Mata Kuliah:** Software Quality  
**Pertemuan:** 12  
**Metode:** Gray Box Testing (Kombinasi White Box & Black Box)

## Definisi
Gray Box Testing adalah metode pengujian yang menggabungkan Black Box dan White Box Testing. Tester memiliki pengetahuan terbatas tentang struktur internal sistem.

## Model yang Diuji
| No | Model | Keterangan |
|----|-------|-------------|
| 1 | Orthogonal Array Testing (OAT) | Menguji kombinasi input dengan jumlah test case minimal |
| 2 | Matrix Testing | Menguji interaksi antar parameter |
| 3 | Regression Testing | Memastikan perubahan kode tidak merusak fungsionalitas lama |
| 4 | Pattern Testing | Eksplorasi bug secara kreatif |
